1.1 Determine limit and the average rate of change

Posted on 2018-01-31

Related section in the textbook: 2.1, 2.2, 2.4
Difficulty: 1/10 (The greater the value, the harder the section)

Important points:
The most direct way to determine a function’s limit is to substitute the c value to which the variable x approaches. However, you need to make sure that the function is simplified and the c value is in the domain of the function.

Example 1.1.1:
Determine lim┬(x→1)⁡〖(x^2-1)/(x-1)〗.

Solution:
Substitute x=1 to f(x)= (x^2-1)/(x-1) and get result 0/0, which is mathematically meaningless. However, if you simplify this function to f(x)=x+1, you will get the correct answer 2.

Tip: Remember the formula x^n-1=(x-1)(x^(n-1)+x^(n-2)…1). It will be helpful in some questions.

Note if the limit is two-sided. You especially need to pay attention to this when determining the limit of a piece-wise function.

Example1.1.2
Consider the following function:

f(x)={█(tan⁡x,x>0@x^2-1,x≤0)┤
Determine lim┬(x^+→0)⁡〖f(x)〗.
Determine lim┬(x→0)⁡〖f(x)〗.

Solution:
Choose the value and function you need to substitute. Here the variable x approaches from positive value so we choose x=0 and f(x)= tan⁡x in this situation.
Therefore lim┬(x^+→0)⁡〖f(x)〗= tan⁡0=0.
Now we have determine lim┬(x^+→0)⁡〖f(x)〗= tan⁡0=0. Using same substitution method, we determine that lim┬(x^-→0)⁡〖f(x)〗=x^2-1 =-1. Therefore lim┬(x^+→0)⁡〖f(x)〗≠lim┬(x^-→0)⁡〖f(x)〗, meaning that the limit lim┬(x→0)⁡〖f(x)〗 does NOT exist.

Warning:
All the angles in calculus are expressed in radian rather than degree.

Sandwich theorem is graphically obvious but important in determining limit before you learn L′ Hospital rule. Usually Sandwich theorem is used when determining the limit of a function consists of trigonometric function and polynomial function.

The end behavior model of a function is essentially another simpler function (simpler means there are fewer terms) that indicates the growth pattern of the original function when the independent variable approaches infinity. (No matter positive or negative infinity)

Example 1.1.3:
Find an end behavior model for function f(x)=cos(x)+x.

Solution:
Let x→∞, then lim┬(x→∞)⁡〖f(x)〗=lim┬(x→∞)⁡〖cos⁡(x)〗+lim┬(x→∞)⁡x. Very obviously the limit lim┬(x→∞)⁡x far exceeds lim┬(x→∞)⁡〖cos⁡(x)〗. So we could ignore the part cos(x). Therefore an end behavior function for f(x) can be any function with form g(x)=x+C (C is a constant). An example can be g(x)=x+1.

Tip: As for finding an end behavior model, you need to separate the function into several part based on the types of the terms. Then find which part has For instance, for f(x), x is a polynomial function and cos(x) is a trigonometric function. However, usually the tests or quizzes only ask you to determine if a function is the end behavior model of a function. Just let the independent variable approach positive or negative infinity. 

The tangent line is a special form of secant line. The secant line has two intersections on a (part of) curve. And the tangent line forms when the two intersections reclose.
